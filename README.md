# 💼 Job Recommendation System

## 📌 About the Project
This is a full-stack Job Recommendation System that suggests jobs based on user skills.  
Users can enter a skill (like Python, React, Java) and get matching job roles.

This project simulates a real-world job portal like LinkedIn or Naukri.

---

## 🚀 Features
- 🔍 Search jobs by skill
- 📄 Real-time filtering
- ⚡ Fast API response
- 🌐 Full Stack (Frontend + Backend)

---

## 🛠️ Tech Stack
- **Frontend:** React.js  
- **Backend:** Python (Flask)  
- **API:** REST API  
- **Data:** JSON
